# Portfolio Card - Neural Debris Hermes Agent

## Title

Neural Debris Hermes Agent - Agentic Kaggle Research

## Short Description

Kaggle research workspace for repairing a poisoned streak detector with a
VPS-hosted Hermes agent, WhatsApp supervision, no-submit experiment gates and
machine-readable project state.

## Longer Modal Text

I built an agentic Kaggle workflow around the ESA Neural Debris challenge. The
goal is to repair a poisoned RetinaNet-style detector while keeping leaderboard
experiments reproducible and controlled. The setup connects local Codex work,
a dedicated Hermes profile on a VPS, Kaggle CLI, WhatsApp prompts and persistent
knowledge files.

The project moved from an early `257.x` public-score plateau to a best observed
account score of `245.3014`, while preserving strict rules: no automatic
submission, no paid GPU without approval, and every external notebook treated as
a hypothesis to audit before reuse.

## Stack

Markdown research reports, JSON runtime state.
